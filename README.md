# Reaching_Synch
This is an analysis code used in a paper "Cooperation Enhances Social Presence Despite Reduced Movement Synchrony."
The following synchrony measures are computed from time serise of two particpants' movement data.
You need to install the "sklearn" and "pingouin" packages to use this code. Some other basic packages are also required. Please check imported packages in the code.

Synchrony measures used in this analysis are:
 Pearson’s correlation
 Cross-correlation
 Coherence analysis
 Phase-Locking Value (PLV)
 Rho ($\rho$) in Circular Correlation
 Mutual information
